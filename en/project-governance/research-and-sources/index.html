    Research and sources¶

    This page collects the core evidence base used across the documentation so readers can inspect the underlying institutions, standards, and research directly.

    Current source library

    This page supports the current docs set and is designed to grow as more implemented features demand tighter technical sourcing.

    How to use this library¶

    The docs use inline citations so readers can check the supporting sources without leaving the page permanently. This library collects the core references in one place so contributors and evaluators can review the evidence base directly.

    Core source groups¶

    How the source library is organized¶

    Group Examples Used for
    Cultural and regional context UNESCO, archaeological research on Georgian wine. Explaining why origin matters in the pilot context.
    Traceability and counterfeit risk FAO and OECD/EUIPO materials. Explaining the practical trust problem.
    Technical reference patterns EAS and OpenZeppelin documentation. Framing attestation and role-control concepts.
    Project governance and openness DPG Standard, Apache 2.0, DCO, NIST SSDF. Explaining reuse, governance, and secure delivery expectations.
