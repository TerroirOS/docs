# რეფერენს აპის დაყენება

Reference app-ის გვერდი დღეს უნდა წავიკითხოთ როგორც მომავალი ოპერაციული აპლიკაციის კონტური.

!!! warning "reference app concept"
    producer/certifier reference app ამ რეპოზიტორიაში ჯერ არ არსებობს.

## რას ნიშნავს reference app

Reference app უნდა იყოს ოპერაციული ფენა, სადაც producers ქმნიან ჩანაწერებს, issuers გასცემენ claim-ებს და იქმნება verification view. ამჟამინდელი რეპოზიტორია კი public website-ს ფარავს.
