# რა არის Terroir

Terroir არის ღია ინფრასტრუქტურის კონცეფცია და დოკუმენტაციის სივრცე სანდო აგრარული წარმომავლობისთვის.

!!! info "პროექტის მიმოხილვა"
    ამ რეპოზიტორიაში დღეს ძირითადად საიტი და დოკუმენტაციაა; ფართო პლატფორმა დოკუმენტებში reference მოდელადაა აღწერილი.

## მოკლედ

Terroir არის ღია კოდის პროტოკოლი და პლატფორმის კონცეფცია აგრარული provenance-ისთვის. მისი მიზანია მცირე მწარმოებლებს, კოოპერატივებსა და სერტიფიკატორებს ჰქონდეთ უფრო სანდო გზა წარმოშობის, პროცესისა და ხარისხის დასადასტურებლად.

პროექტში თითოეული პარტია პირველ კლასის ციფრულ ობიექტად მოიაზრება: ვინ აწარმოა, სად, როდის, როგორ დამუშავდა და ვინ რა დაადასტურა. კრიტიკული მტკიცებულება უნდა იყოს tamper-evident, ხოლო ადამიანისთვის წაკითხვადი ნაწილი — მარტივი და გამოსადეგი.

დღევანდელ რეპოზიტორიაში წარმოდგენილია საიტი, კვლევის გვერდები, თეთრი ნაშრომი და დოკუმენტაციის ბიბლიოთეკა. უფრო ფართო ოპერაციული პლატფორმა ამ ეტაპზე დოკუმენტური reference მიმართულებაა.
