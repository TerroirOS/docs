# არქიტექტურის მიმოხილვა

არქიტექტურა უნდა განვასხვავოთ ორ ნაწილად: ის, რაც საიტში უკვე არსებობს, და ის, რაც მომავალ ოპერაციულ ფენადაა აღწერილი.

!!! info "საიტი + reference არქიტექტურა"
    ეს გვერდი მკაფიოდ ჰყოფს მიმდინარე ვებსაიტს და მომავალ provenance არქიტექტურას.

## ორ ფენად დანახვა

დღეს ამ რეპოზიტორიაში არსებობს public website ფენა: marketing გვერდები, კვლევა, თეთრი ნაშრომი და დოკუმენტაცია. უფრო ფართო სისტემა — batch-ები, attestations, verification სერვისები, მონაცემთა საცავი და integrity ფენა — ამ ეტაპზე reference architecture-ადაა აღწერილი.

ამ განსხვავების ხაზგასმა აუცილებელია, რათა დოკუმენტაციამ ნდობა გაზარდოს და არა პირიქით. კარგი არქიტექტურული დოკუმენტი ერთდროულად უნდა აჩვენებდეს ხედვასაც და მიმდინარე რეალობასაც.
