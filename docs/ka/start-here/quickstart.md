# სწრაფი დაწყება

სწრაფი დაწყების რეალური ნაწილი ეხება ამ საიტს; ფართო პლატფორმის ნაწილი კი reference ჩანაწერია.

!!! info "რეალური საიტის დაწყება + reference ჩანაწერი"
    ამ გვერდზე ერთად არის აღწერილი საიტის ფაქტობრივი გაშვება და მომავალი სრული stack-ის მოლოდინები.

## ამ რეპოზიტორიის დაწყება

თუ თქვენი მიზანია საიტზე და დოკუმენტაციაზე მუშაობა, რეალური setup მარტივია: root დირექტორიაში dependencies-ის დაყენება, `npm run dev`-ით საიტის გაშვება და `npm run lint`-ით ვალიდაცია.

Quickstart-ის სხვა ნაწილი აღწერს უფრო დიდ provenance stack-ს — მონაცემთა ფენას, attestations-სა და verification სერვისებს — მაგრამ ეს ყველაფერი ჯერ reference მიმართულებაა და ამ კოდბაზაში განხორციელებული არ არის.
