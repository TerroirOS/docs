# მართვის როუდმაპი

Governance roadmap აღწერს როგორ შეიძლება პროექტი გუნდის მიერ მართულიდან უფრო ფართო stewardship მოდელამდე მივიდეს.

!!! info "public roadmap"
    ეს გვერდი აერთიანებს პროექტის მიმართულებასა და მიმდინარე დოკუმენტაციურ მდგომარეობას.

## რატომ არის governance მნიშვნელოვანი

Provenance ინფრასტრუქტურას სჭირდება მმართველობა, რადგან schema, issuer authority და verification language მხოლოდ ტექნიკური გადაწყვეტილებები არ არის. თუ ეს წესები გაუმჭვირვალეა, სისტემის ნდობა მცირდება.
