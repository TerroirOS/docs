# ლიცენზირება

ლიცენზირება ხსნის რა პირობებით შეიძლება კოდის გამოყენება, მოდიფიკაცია და ხელახალი გავრცელება.

!!! tip "current licensing guidance"
    ეს გვერდი აღწერს open-source reuse-ის ჩარჩოს, რომელსაც პროექტის ისტორია ეყრდნობა.

## რას ნიშნავს open licensing

ღია ლიცენზია პრაქტიკულ კითხვას პასუხობს: რა შეუძლია სხვა ადამიანს ან ორგანიზაციას ამ კოდთან დაკავშირებით? Terroir-ისთვის ეს განსაკუთრებით მნიშვნელოვანია, რადგან openness მისი ნდობის ნარატივის ნაწილია.
