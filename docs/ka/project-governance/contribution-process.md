# კონტრიბუციის პროცესი

Contribution process ეხმარება ცვლილებების შეტანას ისე, რომ დოკუმენტაცია და რეალური იმპლემენტაცია ერთმანეთს არ დაშორდეს.

!!! tip "current process guidance"
    ეს გვერდი აღწერს როგორ უნდა მიუახლოვდნენ კონტრიბუტორები trust-sensitive documentation ცვლილებებს.

## ძირითადი პრინციპები

კონტრიბუციისას მთავარი წესია: documentation არ უნდა უსწრებდეს repo-ს რეალობას ისე, რომ მკითხველს უკვე არსებული ფუნქციონალის შთაბეჭდილება შეუქმნას. კვლევაზე დაფუძნებული მტკიცებები authoritative წყაროებით უნდა იყოს გამყარებული, ხოლო plain-language summaries — შენარჩუნებული.
